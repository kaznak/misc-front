# Tailwind CSS starter

Features:

- Next.js
- Typescript
- TailwindCSS
- Development Tools
  - prettier
  - eslint
  - jest / react-testing-library

Next.js adopts CSS Modules instead of CSS-in-JS[^1]. Therefore this starter does not includes emotion.js.

[^1]: https://zenn.dev/takepepe/scraps/6668e9fe402666
