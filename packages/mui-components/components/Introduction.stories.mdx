# Atomic Design

The Atomic Design concept is useful to construct component libraries.

- [atomic design(blog post)](https://bradfrost.com/blog/post/atomic-web-design/)
- [Atomic Design(the book)](https://atomicdesign.bradfrost.com/)

5 distinct levels in atomic design:

1. Atoms
2. Molecules
3. Organisms
4. Templates
5. Pages
